# powerbidashboard
This Power BI dashboard analyzes customer churn data to identify key factors influencing customer retention. The goal is to help stakeholders understand churn patterns and make data-driven decisions to reduce customer loss.
Objectives:
Analyze customer churn trends
Identify high-risk customer segments
Provide actionable insights for business decision-making
Database information:
Source: Telecom customer dataset
Format: CSV
Key fields: Customer ID, Tenure, Contract , Monthly Charges, Churn Status
Dashboard Features:
KPI cards showing total customers, churn rate, and retained customers
Filters/slicers for churn and contract type
Visuals such as bar charts, pie charts, line charts, and tables
Drill-down and interactive analysis
